
# Verwendung

Zur Verwendung können die folgenden Variablen der URL hinzugefügt werden

URL: `https://www.nichthallo.de`

Domain auf der Seite einblenden

`?domain=Beispiel.de`

Signatur ausblenden

`?signature=0`

Fußzeile ausblenden

`?footer=0`


## Beispiel

`https://www.nichthallo.de?domain=Beispiel.de&signature=0&footer=0`
